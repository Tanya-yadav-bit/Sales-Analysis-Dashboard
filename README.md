<h1 align="center">📊 Sales Analysis Dashboard (Power BI)</h1>

<h2>📌 Project Overview</h2>
<p>
This project presents an <b>interactive Sales Analysis Dashboard</b> built using Power BI on the Superstore dataset. 
The dashboard provides insights into <b>sales performance, profit trends, and customer behavior</b> 
to support data-driven decision-making.
</p>

<p><b>This project demonstrates skills in data cleaning, exploratory data analysis (EDA), and business insight generation.</b></p>

<hr>

<h2>📂 Dataset</h2>
<ul>
  <li><b>Source:</b> 
    <a href="https://www.kaggle.com/datasets/vivek468/superstore-dataset-final" target="_blank">
      Superstore Dataset (Kaggle)
    </a>
  </li>
  <li><b>Description:</b> Contains data related to orders, sales, profit, region, category, and customer segments.</li>
</ul>

<hr>

<h2>🛠️ Tools & Technologies</h2>
<ul>
  <li>Power BI</li>
  <li>Data Cleaning & Transformation</li>
  <li>Data Visualization</li>
</ul>

<hr>

<h2>📈 Dashboard Features</h2>
<ul>
  <li>KPI Cards (<b>Total Sales, Profit, Quantity</b>)</li>
  <li>Region-wise Sales Analysis</li>
  <li>Category & Sub-category Performance</li>
  <li>Monthly Sales Trends</li>
  <li>Profit vs Sales Analysis</li>
</ul>

<h2>🔍 Key Insights</h2>
<ul>
  <li>West region generated the highest sales</li>
  <li>Technology category contributed maximum profit</li>
  <li>Some sub-categories showed consistent losses</li>
  <li>Seasonal trends observed in monthly sales</li>
</ul>
